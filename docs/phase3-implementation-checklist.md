## 文档目的

将第三阶段“动态图能力稳定化”目标拆解为可执行清单。  
本清单默认全部为必须项（Must），不再区分可选项。

## 后端实施清单（Must）

- [ ] 提供动态图任务创建接口
- [ ] 提供动态图任务状态查询接口
- [ ] 供应商适配层支持至少 1 种云服务与 1 种第三方服务扩展位
- [ ] 生成结果必须中转上传到自有存储
- [ ] 返回给小程序的图片引用必须为自有域名 URL 或 file_id
- [ ] 增加幂等缓存与失败重试
- [ ] 增加动态生图开关与故障降级

## 小程序实施清单（Must）

- [ ] 结果页只消费后端统一图片引用
- [ ] 不直接依赖第三方图片域名
- [ ] 展示图片生成状态（生成中/失败/完成）
- [ ] 远端图片失败时自动回退本地静态图
- [ ] 兜底逻辑不影响主结果文本展示

## 数据与安全清单（Must）

- [ ] 明确生成图生命周期与清理策略
- [ ] 清理任务可回归验证
- [ ] 体验版/正式版域名策略一致
- [ ] 文档明确禁止端侧直连第三方动态图地址

## 验收清单（Must）

- [ ] iOS 真机体验版：诗人/国潮图稳定展示
- [ ] Android 真机体验版：诗人/国潮图稳定展示
- [ ] 5G 弱网下可回退本地静态图，不出现长期空白
- [ ] 供应商超时/失败时，主分析结果不被阻塞
- [ ] 页面与邮件图片一致性可验证

## 封板标准

仅当本清单全部勾选完成，第三阶段才可封板。
