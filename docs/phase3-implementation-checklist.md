## 后端实施清单（Must）

- [ ] 建立静态图池元数据结构（字段完整：`id,url,style,emotion_tags,intensity,active,weight,updated_at`）
- [ ] 后端支持按情绪语境筛选候选素材
- [ ] 后端支持加权随机选图
- [ ] 后端支持同用户近 7 天去重
- [ ] 图片输出统一为腾讯云存储可访问 URL 或 file_id
- [ ] 图池命中失败可按固定顺序回退（风格候选池 -> 通用图池 -> 本地默认图）
- [ ] 主分析结果不受图片链路失败影响
- [ ] 移除第三方动态生图调用代码（qwen/hunyuan/liblib/http）
- [ ] 配置与文档不再出现第三方生图参数误导

## 小程序实施清单（Must）

- [ ] 结果页只消费后端统一图片引用
- [ ] 保留并统一“加载中/成功/失败/兜底”状态提示
- [ ] 失败时回退图与提示文案一致
- [ ] 诗词区与国潮区均可独立展示图片状态
- [ ] 首页/结果页/分享页视觉继续优化（信息层级、主次按钮）

## 数据与安全清单（Must）

- [ ] 体验版/正式版域名白名单配置一致
- [ ] 云存储素材目录权限可控（只读公开或受控访问）
- [ ] 素材可上下线（active 标记）
- [ ] 素材 URL 有失效检测与兜底策略
- [ ] 每周补图机制落地（固定节奏）
- [ ] 权重调优机制落地（依据点击/收藏/邮件打开）
- [ ] 调权记录可追踪（支持回归核对）

## 验收清单（Must）

- [ ] iOS 真机：科技/国潮图片稳定展示
- [ ] Android 真机：科技/国潮图片稳定展示
- [ ] 5G 弱网：不出现长期空白图片区
- [ ] 页面与邮件图片来源一致可追踪
- [ ] 图片失败时不阻塞主分析结论展示
- [ ] 无文字重叠、无遮挡、无异常滚动抖动

## 封板标准

仅当本清单全部勾选完成，第三阶段才可封板。
